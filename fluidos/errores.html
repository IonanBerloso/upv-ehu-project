Exámenes Errores Frecuentes

⚠️ Errores Frecuentes — Mecánica de Fluidos UPV/EHU

Errores reales detectados en exámenes de convocatorias 2020–2025. Incorrecto (rojo) → Correcto (verde).

💧 Hidrostática
✗ Error

Usar profundidad de la base en F = P̄·A

✓ Correcto

ȳ = profundidad del centroide de la superficie

✗ Error

F_V curva = fluido encima cuando está debajo

✓ Correcto

Fluido debajo → F_V hacia arriba = γ·V_ficticio

✗ Error

Confundir 1 atm = 1 bar

✓ Correcto

1 atm = 101 325 Pa = 1,013 bar = 10,33 mca

✗ Error

En flotación: E = γ·V_total siempre

✓ Correcto

E = γ_fluido·V_sumergido (no necesariamente todo el cuerpo)

✗ Error

ΔP burbuja de jabón = 2σ/R

✓ Correcto

Jabón (2 interfaces): ΔP = 4σ/R · Burbuja en líquido: 2σ/R

🌀 Bernoulli
✗ Error

Aplicar Bernoulli entre dos puntos de líneas de corriente distintas

✓ Correcto

Solo válido en la misma línea de corriente

✗ Error

En depósito grande: v_superficie ≠ 0

✓ Correcto

Depósito grande: v ≈ 0 y P = P_atm en la superficie libre

✗ Error

Sumar h_f al lado de energía de salida

✓ Correcto

h_f siempre resta energía en el sentido del flujo: E₁ + Hm = E₂ + h_f

✗ Error

No usar continuidad antes de Bernoulli → 2 incógnitas

✓ Correcto

Siempre: continuidad primero → v₂ = v₁·(A₁/A₂), luego Bernoulli

💥 Cantidad de Movimiento
✗ Error

Olvidar los términos P·A en tuberías a presión

✓ Correcto

En QdM con presión: incluir P₁A₁ (entrada) y −P₂A₂ (salida) en ΣF

✗ Error

F sobre el fluido = F sobre la placa

✓ Correcto

F_placa = −F_fluido (3.ª ley de Newton)

✗ Error

Chorro oblicuo: no descomponer en x e y

✓ Correcto

Proyectar siempre. QdM es vectorial → ecuación por cada eje

🔧 Tuberías y Colebrook
✗ Error

No iterar en Tipo II/III: usar f inicial como resultado final

✓ Correcto

Siempre iterar hasta error < 1 %. Mínimo 2–3 iteraciones en examen

✗ Error

ε/D con ε en mm y D en m (o al revés)

✓ Correcto

ε y D en las mismas unidades. ε/D es adimensional

✗ Error

Tipo III: escoger diámetro comercial inferior al calculado

✓ Correcto

Diámetro comercial superior: para que el caudal real ≥ caudal de diseño

✗ Error

En ensanchamiento brusco: h = (v₁²−v₂²)/2g

✓ Correcto

Borda-Carnot: h = (v₁ − v₂)²/2g  [diferencia primero, luego elevar al cuadrado]

⚙️ Bombas y NPSH
✗ Error

Olvidar Δz en la curva del sistema

✓ Correcto

H_sist = Δz + K·Q². El Δz es la altura geométrica entre depósitos

✗ Error

NPSH: no incluir h_f en aspiración

✓ Correcto

NPSH_d = (P_atm−P_v)/γ − z_b − h_f,asp

✗ Error

Bombas en paralelo: sumar H a igual Q

✓ Correcto

Paralelo: sumar Q a igual H. Serie: sumar H a igual Q

✗ Error

Usar γ = 9 810 N/m³ para agua

✓ Correcto

UPV/EHU: g = 9,8 → γ_agua(20°C) = 9 782 N/m³. No usar 9 810

🌊 Canales y Manning
✗ Error

J en % en vez de m/m (adimensional)

✓ Correcto

J = pendiente del fondo en m/m. Si te dan 1 ‰ → J = 0,001

✗ Error

R_h = D/4 siempre

✓ Correcto

R_h = D/4 solo para tubería circular llena. En canal: R_h = A/P_m

✗ Error

Confundir y_c (crítico) con y_n (normal)

✓ Correcto

y_n → Manning (flujo uniforme). y_c → Froude = 1 (flujo crítico). Son distintos

🔢 An. Dimensional
✗ Error

Repetidas que ya forman un grupo adimensional entre sí

✓ Correcto

Verificar que las k repetidas son dimensionalmente independientes entre sí

✗ Error

Aplicar semejanza Reynolds a canal abierto

✓ Correcto

Canal abierto (gravedad domina) → semejanza Froude. Reynolds → tuberías a presión

📏 Medida de caudal
✗ Error

Δh de manómetro = lectura T directamente

✓ Correcto

Δh = T·(s_manómetro/s_fluido − 1). Si ambos son agua: Δh = T

✗ Error

Q_venturi = A₂·√(2gΔh)

✓ Correcto

Q_teo = (A₁A₂/√(A₁²−A₂²))·√(2gΔh). No se puede simplificar a solo A₂

🎯 Checklist examen — antes de entregar
▸g = 9,8 m/s² en todos los cálculos. No usar 9,81
▸Unidades coherentes: P en Pa, L en m, Q en m³/s. No mezclar
▸Continuidad antes de Bernoulli. Siempre
▸Iterar Colebrook hasta error <1 %. Mínimo 2 iteraciones
▸Diámetro comercial superior en Tipo III
▸Signos QdM: definir eje positivo antes de proyectar
▸NPSH: incluir h_f en aspiración. z_b positivo si bomba está por encima
▸Manómetro: Δh = T·(s_m/s_f − 1), no T directamente
▸ε/D adimensional: ε y D en las mismas unidades
▸P_v en Pa para NPSH. No confundir con P_v en bar o mca
